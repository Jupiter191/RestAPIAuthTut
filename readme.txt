This project follows the tutorial located at
"https://medium.freecodecamp.org/securing-node-js-restful-apis-with-json-web-tokens-9f811a92bb52"

This tutorial involves adding user authentication and authorization to the rest API created in the previous tutorial located at:
"https://hackernoon.com/restful-api-design-with-node-js-26ccf66eab09"

NOTES
When getting this repo working locally
- Copy db_example.js
- Rename to db.js
- Replace placeholder mLab database connection with your own

- Copy config_example.js
- Rename to config.js
- Replace placeholder secret and other variables with your own
